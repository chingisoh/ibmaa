Question 1 <br>

Let \(f(x)=\mathrm{e}^{x+3}\).

<div class="q-part">
  <span class="label">(a)</span>

  <div class="subpart">
    <span class="sublabel">(i)</span>
    Show that \(f^{-1}(x)=\ln x-3\).
  </div>

  <div class="subpart">
    <span class="sublabel">(ii)</span>
    Write down the domain of \(f^{-1}\).
  </div>

</div>

<div class="q-part">
  <span class="label">(b)</span>
  Solve the equation \(f^{-1}(x)=\ln \left(\frac{1}{x}\right)\).
</div>

<div id="sol1" class="solution" hidden>
<b>Solution:</b>

(a)(i) <br>
Let \( y=e^{x+3}\)<br>
\(x=e^{y+3}\)<br>
\(\ln x = y+3\) <br>
\(y=\ln x -3\)<br>
\(\therefore f^{-1}(x)=\ln x -3\) <br><br>

(a)(ii) <br>

\(x > 0\)<br>

<span class="blue"> CONCEPT! For \(\log_a x\) to be defined, \(x > 0\).</span><br>

(b) <br>

\(\ln x -3=\ln \left(\dfrac1x\right)\)<br>

\(\ln x - 3 = -\ln x\) <br>

\(2\ln x = 3\) <br>

\(\ln x =\dfrac32\) <br>

\(x=\mathrm e^{\frac32}\)<br>

<span class="blue"> CONCEPT! Logarithms and exponentials are inverse functions.</span><br>
